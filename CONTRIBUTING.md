# Contributing to payment_athmovil

Thank you for your interest in contributing. This module is maintained by **Spark Workforce LLC**.

## Before You Start

- Read the [README](payment_athmovil/README.md) to understand the module's purpose and architecture
- Review the [AI-DLC documentation](aidlc-docs/) for design decisions and requirements
- Check existing [issues](../../issues) before opening a new one

## Reporting Issues

When reporting a bug, include:
- Odoo version (17, 18, or 19)
- Deployment type (Odoo.sh, on-premises, Docker)
- Steps to reproduce
- Expected vs actual behavior
- Relevant log output (never include your ATH Business tokens)

## Pull Requests

1. Fork the repository
2. Create a branch: `git checkout -b fix/your-fix-name` or `feature/your-feature-name`
3. Follow Odoo coding standards: [Odoo Guidelines](https://www.odoo.com/documentation/17.0/contributing/development/coding_guidelines.html)
4. Add inline comments in English for any non-obvious logic
5. Update `CHANGELOG.md` under `[Unreleased]`
6. Open a pull request against `main`

## Security Issues

Do **not** open public issues for security vulnerabilities. Email [jvelez@sparkworkforcellc.com](mailto:jvelez@sparkworkforcellc.com) directly.
