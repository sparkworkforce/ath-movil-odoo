# payment_athmovil — ATH Móvil Payment Provider for Odoo

Integrate [ATH Móvil Business](https://ath.business) as a native payment method in Odoo 17, 18, and 19.

ATH Móvil is Puerto Rico's dominant mobile payment network, operated by EVERTEC Group, LLC.
