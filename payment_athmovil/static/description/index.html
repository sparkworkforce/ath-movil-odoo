Easy Setup / Configuración fácil

  1. Install the module / Instala el módulo
  2. Go to Settings → Payment Providers → ATH Móvil / Ve a Ajustes → Proveedores de Pago
  3. Follow the step-by-step guide / Sigue la guía paso a paso
  4. Paste your tokens and click "Test Connection" / Pega tus tokens y haz clic en "Probar Conexión"
  5. Enable the provider — you're live! / ¡Activa el proveedor y listo!
